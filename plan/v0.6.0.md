Implement the following features:

* Add a `required` property to mark the field as mandatory.
* Add support for a `validator` function.
* Add a `disabled` property to disable user interaction with the field.
* Add support for assigning a custom theme directly to the field.
* Add `focusedStyle` to `AutoSuggestionsBoxThemeData`, allowing customization of focused-state styling, including:

  * `fillColor`
  * `border`
  * `fontStyle`
  * Any other relevant visual properties.
